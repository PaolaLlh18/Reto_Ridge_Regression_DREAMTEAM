## Descripción
Este proyecto implementa un sistema de reconocimiento de expresiones faciales utilizando **Ridge Regression** y la geometría de un **símplex regular**, basado en el paper de An, Liu & Venkatesh (2007).

* `ML Ridge Caras An07.ipynb`: Código principal con el pipeline de entrenamiento y pruebas.
* `expresiones.jpg`: Dataset original de expresiones faciales.
